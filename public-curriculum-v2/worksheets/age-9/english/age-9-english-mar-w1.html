Deep Character Analysis

March Week 1

Age 9

Literature Study
🎯 Learning Goal

Analyze characters using text evidence: describe traits, motivations, feelings, and how characters change throughout a story.

☀️ Warm-Up: Character Emoji

Name a character from a book you've read. Pick an emoji that represents how that character felt at the END of the story. Explain why using one specific event.

📝 Activity: Character Analysis Map & STEAL

Create a Character Analysis Map with four sections:

  • Outside Traits: Physical description from the text
  • Inside Traits: Personality — find text evidence
  • Motivations: What does the character want and why?
  • Changes: How does the character grow from beginning to end?

Use STEAL: What the character Says, Thinks, Effects on others, Actions, Looks.

Part C: Character Traits — Match the trait to the evidence.

  1. "She shared her lunch with the new student who had none." → Generous / Brave / Curious
  2. "He stood up to the bully even though he was scared." → Honest / Courageous / Patient
  3. "She practiced piano every single day without anyone telling her to." → Lazy / Self-disciplined / Shy
  4. "He always told the truth, even when it got him in trouble." → Funny / Honest / Creative

Part D: Write one paragraph analyzing how a character changed from the beginning to the end of a story.

✍️ Writing Space
✅ Can Your Child...?
  • Complete a Character Analysis Map with 2+ outside traits, 3+ inside traits with evidence
  • Use the STEAL method to find text evidence
  • Identify a character's main motivation
  • Describe how a character changed with quotes from the text
💡 Easier: Focus on inside traits and motivations only, using a fill-in-the-blank template.
💡 Harder: Compare two characters from the same book using a T-chart.
